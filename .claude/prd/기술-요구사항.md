# 기술 요구사항

**문서 버전**: 1.1
**최종 수정일**: 2025-11-11
**상위 문서**: [PRD](../PRD.md)

---

## 개요

이 문서는 8G 플랫폼의 기능적 요구사항(Functional Requirements)과 비기능적 요구사항(Non-Functional Requirements)을 정의합니다.

---

## 문서 인덱스

이 문서는 200줄 제한 준수를 위해 하위 문서로 분리되어 있습니다.

**상세 문서**:

- [기능 요구사항](./기술-요구사항/기능-요구사항.md) - FR-001~009: 워크플로우 생성/저장, 블록, 조건, 루프, 변수, 실행, 결과, Import/Export
- [비기능 요구사항](./기술-요구사항/비기능-요구사항.md) - NFR-001~007: 성능, 확장성, 유지보수성, 사용성, 보안, 호환성, 가용성

---

## 기능적 요구사항 요약

**FR-001**: 워크플로우 생성 (드래그 앤 드롭, 블록 설정) - P0
**FR-002**: 워크플로우 저장/로드 (MySQL JSON) - P0
**FR-003**: 블록 시스템 (20종+ 블록) - P0
**FR-004**: 조건부 분기 (equals, regex, contains, exists, AND/OR) - P0
**FR-005**: 루프 실행 (forEach, count) - P1
**FR-006**: 변수 시스템 (정적, 동적, 컨텍스트) - P0
**FR-007**: 워크플로우 실행 (Extension 기반) - P0
**FR-008**: 결과 시각화 (JSON/Table/Raw 뷰) - P1
**FR-009**: Import/Export (JSON, CSV) - P1

**상세**: [기능 요구사항](./기술-요구사항/기능-요구사항.md)

---

## 비기능적 요구사항 요약

**NFR-001**: 성능 (페이지 로드 3초, 60fps) - P1
**NFR-002**: 확장성 (블록 자동 생성, 인덱싱) - P1
**NFR-003**: 유지보수성 (TypeScript, ESLint, 200줄) - P1
**NFR-004**: 사용성 (10분 내 생성 가능) - P0
**NFR-005**: 보안 (암호화, SQL Injection 방지) - P1
**NFR-006**: 호환성 (Chrome 90+, Node 20+) - P1
**NFR-007**: 가용성 (99.5% uptime) - P1

**기술 제약사항**:

- TC-001: Extension 의존성
- TC-002: 클라이언트 실행
- TC-003: JSON 크기 제한

**상세**: [비기능 요구사항](./기술-요구사항/비기능-요구사항.md)
